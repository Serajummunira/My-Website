HW6 Q-2: Improvement of Homepage what Features I Added

1. Added Typewriter Effect

This feature displays a typing animation with the text "Welcome to my awesome website". This makes the page feel more dynamic and interactive, especially when the user first loads the site.

2. Added Real-Time Date and Time

The date and time are updated in real-time on the page. This keeps the page content current and gives a more interactive feel for the user, as they can see the time as it changes.

3. Added Toggled Dark Mode

A button is provided to toggle between light and dark mode on the page. Dark mode provides an alternative color scheme that is easier on the eyes, especially in low-light environments.

4. Added Visitor Count

The page keeps track of the number of visitors and displays the count. This is useful for seeing how many people have visited the website, adding a sense of interaction with the page.

5. Added "HW6 is ready for grading" Sparkling Effect

This feature makes the text "HW6 is ready for grading" sparkle when the user hovers over it. It draws attention to the important message, making it stand out as a key notification for the visitor.

6. Added GitHub and LinkedIn for Connecting with Social Media

Links to GitHub and LinkedIn have been added to the page. This allows visitors to connect with you on social media and view your professional and technical work.
